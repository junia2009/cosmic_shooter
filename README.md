# Cosmic Shooter

3D宇宙シューティングゲーム（Three.js使用）

## 概要
- プレイヤーは宇宙空間で敵を撃ち落とし、スコアを競います。
- 制限時間内にできるだけ多くの敵を倒しましょう。
- 敵の種類によって得点が異なります。
- スコアに応じてランクが表示されます。

## 操作方法
- マウスで自機を左右・上下に移動
- クリックまたはスペースキーでショット
- ゲーム終了後「RESTART」ボタンで再挑戦

## 技術要素
- Three.jsによる3D描画
- HTML/CSS/JavaScriptのみで構成
- シングルファイルで動作

## ファイル構成
- cosmic_shooter.html : ゲーム本体
