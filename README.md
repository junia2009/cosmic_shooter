# Cosmic Shooter 3D

Three.js製の3D宇宙シューティングゲームです。

## 特徴
- スタート画面に3Dアニメーション（自機・敵・星）
- ゲーム本編も3Dで動作
- PC/スマホ両対応（タッチ・マウス・キーボード）
- スコア・タイマー・ランク表示

## 遊び方
- スタート画面の「START」ボタンでゲーム開始
- 矢印キーまたはドラッグ/タップで自機移動
- スペースキーまたは画面タップ/クリックでショット
- 制限時間内にスコアを稼ごう！

## 動作環境
- モダンブラウザ（importmap対応、例: Chrome, Edge, Firefox最新版）
- ローカルで動かす場合は httpサーバー経由推奨
  - 例: VSCodeのLive Server拡張、または `python -m http.server`

- index.html ... ゲーム本体（HTML+JS一体型）
